# DDE Spark Portal

Employee recognition system for D. DuBaldo Electric.

## Setup

```bash
npm install
npm run dev
```

## Deploy

This project is configured for GitHub + Netlify auto-deploy.

1. Push to GitHub
2. Connect repo in Netlify dashboard
3. Build command: `npm run build`
4. Publish directory: `dist`

Netlify will auto-deploy on every push to `main`.
